# Sistema Inteligente con Reentrenamiento Continuo para la Detección Adaptativa de Patologías Cardiovasculares Basado en ECG

Este repositorio contiene el desarrollo del anteproyecto y prototipo funcional de un sistema inteligente capaz de detectar patologías cardiovasculares mediante señales ECG y mejorar su precisión con el tiempo gracias a un esquema de reentrenamiento continuo (*continual learning*).

## Objetivo General
Desarrollar un sistema adaptable que optimise la detección de anomalías cardiovasculares mediante algoritmos de aprendizaje automático con reentrenamiento incremental.

## Componentes principales
- Procesamiento de señales ECG.
- Extracción de características.
- Modelos de machine learning y deep learning.
- Arquitectura de reentrenamiento continuo.
- Evaluación con métricas biomédicas.
